个人学习JVM是尚硅谷-书-黑马

不建议直接去看书，书上东西很深，跟着视频理清一下脉络再去看书会更好理解些

看两个视频的原因是一是横向对比，二是对已有的知识进行复习

两个视频各有优劣（个人评价）
- 尚硅谷：55小时的时常，讲的很细致，同时也太过细致，导致认为全是重点，学习起来相对难度较大for me
- 黑马：17小时的时常，简单精干

我更喜欢黑马老师的讲课风格一些，笔记基于黑马，他人博客、个人理解作为补充
